# SAM-TTS-Installer-For-MacOS

Installs S.A.M. (Commodore C64 vocal synthesis program) into terminal.

Works best with MacOS 10.15-14, newer versions may work but older ones will not.

Once you run the app, use "saysam -help" for the info on how to use it.

P.S. It requires wifi to download the SAM program.
